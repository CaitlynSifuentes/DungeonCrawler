# DungeonCrawler

 <p><a href="https://catiesif.itch.io/nightly-duties" target="_blank">Nightly Duties</a></p>
<p>Help your knight clean out the dungeon and fulfill his duties! The weapon shop is only available on the shop floor so be sure to hurry back when you have enough coin!&nbsp;</p>
<p><strong>Controls:</strong></p>
<p>Move:&nbsp; WSAD<br></p>
<p>Left click: Attack</p>
<p>Heal with the healing fountains found on the shop floor or the first dungeon level (Beware as the fountains are not everlasting)</p>
